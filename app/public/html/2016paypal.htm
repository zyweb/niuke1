内推说明
作为PayPal的合作方，牛客拥有10个内推名额，通过者可优先过笔试、优先安排面试、优先录取。牛客提前内推名额未通过将进入正常校招流程。
工作地点
上海
参加要求
2016应届生，计算机相关专业（仅限硕士、博士）。
参加条件
1、登录牛客账号（账号需激活）且完善了个人资料（学校、专业、入学时间），才可提交简历；
2、简历提交后，牛客网将严格按照公司要求进行初步筛选，联系初选通过的同学进行后续内推流程，请保持邮箱畅通。
3、无论是否通过牛客网的初选，参加内推的同学将会在2个工作日内收到牛客网的邮件反馈（注：接收邮箱为牛客网注册邮箱，非简历内邮箱；如没有收到，请检查是否在垃圾箱中，或是否邮箱设置了拒收，为避免错过重要信息，尽量不要使用学校邮箱注册）。
4、之后的流程以具体公司为准，牛客网并不清楚，可通过具体公司的官方渠道查询进度。
内推职位
1、风险工程师
工作内容/职位描述：
1) 管理大数据基础设施和开发分析应用，以提供世界一流的数据驱动风险决策。
2) 编写高可用，易测试，易拓展的代码来实现风险模型
3) 严格审查代码质量
4) 实时监控和优化风险模型
5) 设计、开发和维护小应用和流程来支持更好的分析工具包
6) 与分布在世界不同时区的全球风控工程师高效合作
任职资格：
1) 工程，计算机，数学，统计等相关专业的硕士或以上
2) 良好的沟通能力。中英文读写流利
3) 熟练运用统计模型，算法和数据结构。熟悉主流语言C++ / Java / .NET 之一。
4) Linux及windows操作系统知识和经验，会使用Python / Perl / Ruby 等常见脚本语言之一。
5) SQL经验以及对数据库平台有良好的理解
6) 如果拥有统计软件如R,SAS等经验会是加分项
7) 大数据解决方案如Hadoop的知识会是加分项
2、风险分析师
工作内容/职位描述：
1) 进行深入系统化的定量分析, 以获取深度业务含义，寻找风险的蛛丝马迹
2) 不断改进与完善支付产品的性能与流程处理
3) 设计与实施达到风险-收益最优化的决策
4) 建立针对反欺诈行为的预测模型，并加以优化
5) 建立与维护系统性能监控工具，保证工作的顺利进行
任职资格：
1) 工程，计算机，数学，统计，金融等相关专业的硕士或以上
2) 出色逻辑思维能力，对于数据有一定的敏感性；具有数据分析挖掘或者建模相关知识的应聘者优先考虑
3) 扎实的软件编程能力，熟练运用一种及以上编程语言， 有经验管理大量且复杂的数据(SAS, SQL, Excel/VBA, Excel PivotTables) 者最佳
4) 良好的沟通能力，中英文读写流利。
5) 了解电子商务, 网络支付或者移动平台应用者优先考虑
