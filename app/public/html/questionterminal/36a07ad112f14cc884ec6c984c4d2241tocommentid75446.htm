    这道题我没有作对，于是就亲身实践了一下，做了一下实验，貌似有这样的规律，当两个操作数都是常量时（也不准确，因为const时，也会如前者），当移动位数超过了限界，结果将会为0.即1<<32 = 0;1<<33 = 0;1<<34=0;。。。。。
    当两个操作数中有一个为变量时，将会变成算术左移
    int a =1;
    int b = 32;
    a<<b = 1;a<<33 = 2;1<<b = 1;

    const int a = 1;
    a<<b = 1;
